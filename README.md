# Parte Frontend de la practica integrada grupo 7

## Scripts

En el directorio del proyecto puedes ejecutar:

### `npm start`

Ejecuta la app en modo de desarrollo.
Abre [http://localhost:3000](http://localhost:3000) para visualizar en el navegador.

La pagina se recarga si editas los archivos.
Se veran cualquier error SonarLint en la consola.

### `npm run testselenium`

Ejecuta los tests de selenium de forma recursiva.

### `npm run build`

Compila la app para produccion en la carpeta 'build'.
Optimiza la compilación para obtener el mejor rendimiento

La app esta lista para ser desplegada!
